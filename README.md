# Algorithms and Data Structures
Collection of C++ projects including graph traversal, shortest path search, cycle detection and basic data management.

## Projects
### shortestPath.cpp
Transport route finder based on graph representation and BFS traversal. The program builds a network of stops and connections, then finds the shortest route between selected locations.
### sejfy.cpp
Graph-based safe network solver focused on detecting cycles in a system of safes and keys. The program analyzes dependencies between safes and determines how many safes must be destroyed to break inaccesible cycles.
### bazaDanychKsiazek.cpp
Simple console database manager for storing, searching, deleting, saving and loading records. The project focuses on vector-based data storage, file operations and text search.
